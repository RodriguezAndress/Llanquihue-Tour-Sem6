Llanquihue Tour es un sistema automatizado de una agencia de turismo que busca facilitar el trabajo y evitar errores que puedan ocurrir con el registro manual.
En este caso se aplica el uso de toString y fromString además del registro y lectura de archivo txt, un ArrayList que captura distintos tours con 
parametros como nombre, tipo de tour y duración. Además se maneja la herencia entre clases y el poliformismo. *Prueba formativa semana 6*
